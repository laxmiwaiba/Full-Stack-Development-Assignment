HTML – HyperText Markup Language

The backbone of the web: structure, meaning, and content of every page.

What is HTML?

HTML (HyperText Markup Language) is the foundation of the web. It provides the structure for text, images, links, and media so browsers can render them properly. HTML uses tags like <h1>, <p>, and <img> to define the role of each element.

With HTML for structure, CSS for style, and JavaScript for interactivity, developers create engaging websites.

Example 1: Basic Page Structure

The skeleton of every HTML document:

<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <title>My First HTML Page</title>
</head>
<body>
  <h1>Welcome to My Website</h1>
  <p>This is a simple paragraph.</p>
</body>
</html>
Output:

Welcome to My Website

This is a simple paragraph.

Example 2: Links & Images

Embed images and create clickable hyperlinks:

<h2>My Favorite Website</h2>
<a href="https://www.wikipedia.org">Visit Wikipedia</a>

<h2>My Favorite Image</h2>
<img src="https://via.placeholder.com/200" alt="Sample image">
Output:

My Favorite Website

Visit Wikipedia

My Favorite Image

Example 3: Lists & Tables

HTML can also display structured lists and tabular data:

<h3>Shopping List</h3>
<ul>
  <li>Milk</li>
  <li>Bread</li>
  <li>Eggs</li>
</ul>

<h3>Student Scores</h3>
<table border="1">
  <tr><th>Name</th><th>Score</th></tr>
  <tr><td>Alice</td><td>95</td></tr>
  <tr><td>Bob</td><td>88</td></tr>
</table>
Output:

Shopping List

  • Milk
  • Bread
  • Eggs

Student Scores

NameScore
Alice95
Bob88
